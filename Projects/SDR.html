  • Context and description

    Software-defined radio (SDR) is a radio communication system where components that have been traditionally implemented in hardware (mixers, filters, amplifiers, modulators/demodulators, detectors, etc.) are instead implemented by means of software on a personal computer or embedded system.

    These labs were part of the Protocol and Communication course.

    During these labs, we used a National Instruments USRP-2900 software defined radio transceiver which is connected to a computer via a Gigabit Ethernet connection. In the first lab, we did a mathematical analysis of the signal in order to better comprehend the following works. The second lab was based on the gnuradio-companion software. The aim of this lab was to process a radio signal we studied in the previous lab. The signal contained three radio signals from RFM, Nostalgie and Skyrock, that we transposed, filtered, and demodulated it in order to obtain a listenable audio. Using our implemented chain and a transceiver USRP we could listen to different radios in real time.
